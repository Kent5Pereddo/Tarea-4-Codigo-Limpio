Este es el código de mi tarea para simular un catálogo de videos (como películas, series y documentales). Lo hice en Java para practicar los temas de la Unidad 2, especialmente la herencia y cómo conectar las clases entre sí usando Agregación, Composición y Asociación.

## ¿De qué trata este proyecto?
Este proyecto consiste en la transformación de un sistema de gestión audiovisual incompleto hacia una arquitectura de software profesional y escalable. El objetivo principal es aplicar técnicas de refactorización y principios de programación limpia para corregir errores de compilación y mejorar la calidad del código. A través de este proceso, se busca eliminar el código redundante y asegurar que todas las clases y métodos tengan responsabilidades claramente definidas. La meta final es entregar una aplicación robusta que gestione eficazmente películas, series de TV y documentales. 
La implementación se fundamenta en el patrón de diseño Modelo-Vista-Controlador (MVC) para separar la lógica de negocio de la interfaz de usuario. Se integra además un sistema de persistencia mediante archivos CSV que permite la lectura y escritura de datos de forma eficiente y segura. El diseño del sistema sigue estrictamente los principios SOLID, garantizando que el software sea fácil de mantener y extender en el futuro. Finalmente, el proyecto incluye pruebas unitarias exhaustivas para validar que todas las funcionalidades críticas operen correctamente bajo estándares profesionales.
